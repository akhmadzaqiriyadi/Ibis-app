# IBISTEK UTY Landing Page & Ecosystem

Modern landing page and platform for **Inkubator Bisnis dan Teknologi (IBISTEK)**.

## Project Structure

- **frontend/**: Next.js 16 (App Router), Tailwind CSS v4, TypeScript.
- **backend/**: (Coming Soon) API services.

## Getting Started (Frontend)

1. Navigate to frontend:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```

## Features

- **Modern Design**: Premium UI with Tailwind CSS v4 and Clean Light Theme.
- **Animations**: GSAP-powered entrance animations and smooth scrolling.
- **Responsive**: Mobile-first design for all device sizes.
- **Atomic Design**: Modular component architecture.
